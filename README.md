This is a classic Pong game implemented using Python and Pygame.

## Controls
- Left Paddle: W / S
- Right Paddle: Up / Down Arrow Keys

## Features
- Two-player paddle control
- Ball collision with paddles and walls
- Smooth gameplay at 60 FPS

## How to Run
1. Install pygame:
   pip install pygame
2. Run the game:
   python main.py
